## Story
**As a** paciente autenticado,
**I want** enviar mensagens de texto para o concierge e receber respostas organizadas,
**so that** possa obter orientações de saúde e agendar consultas de forma conversacional.

## Acceptance Criteria

1. A rota `/chat` é protegida: usuário sem sessão em `sessionStorage` é redirecionado para `/login`
2. A página exibe o histórico de mensagens com distinção visual entre mensagens do usuário e do concierge
3. Há um campo de texto na parte inferior da tela para digitar a mensagem
4. Há um botão "Enviar" que fica desabilitado enquanto o campo está vazio ou uma requisição está em andamento
5. Ao enviar, a mensagem do usuário aparece imediatamente no histórico (otimistic UI)
6. O frontend chama `POST /api/atendimento/interagir` enviando `Prompt`, `JsonHistory` (histórico serializado como JSON string), `PatientId` e `PatientName`
7. A resposta do backend é renderizada como mensagem do concierge no histórico (`response.resposta`)
8. Durante o carregamento da resposta, um indicador de "digitando..." é exibido
9. O histórico de mensagens é mantido em estado React (`useState`) e atualizado a cada interação
10. O scroll da conversa desce automaticamente para a mensagem mais recente
11. Em caso de erro da API, uma mensagem amigável é exibida no chat ("Desculpe, ocorreu um erro. Tente novamente.")
12. A interface é adequada para público idoso: fontes grandes, balões de mensagem espaçosos, alto contraste

## Dev Notes

### `src/hooks/useChatHistory.ts`

```typescript
import { useState } from 'react';
import { Message, PatientSession } from '@/types';
import { sendTextMessage } from '@/services/api';

export function useChatHistory() {
  const [messages, setMessages] = useState<Message[]>([]);
  const [isLoading, setIsLoading] = useState(false);

  const addMessage = (msg: Message) =>
    setMessages(prev => [...prev, msg]);

  const sendMessage = async (text: string, session: PatientSession) => {
    const userMsg: Message = {
      role: 'user',
      content: text,
      timestamp: Date.now(),
      type: 'text',
    };
    // Optimistic UI: adiciona a mensagem do usuário imediatamente
    const updatedMessages = [...messages, userMsg];
    setMessages(updatedMessages);
    setIsLoading(true);

    try {
      const response = await sendTextMessage(text, updatedMessages, session);
      const assistantMsg: Message = {
        role: 'assistant',
        content: response.resposta,
        timestamp: Date.now(),
        type: 'text',
      };
      setMessages(prev => [...prev, assistantMsg]);
    } catch {
      const errorMsg: Message = {
        role: 'assistant',
        content: 'Desculpe, ocorreu um erro ao processar sua mensagem. Por favor, tente novamente.',
        timestamp: Date.now(),
        type: 'text',
      };
      setMessages(prev => [...prev, errorMsg]);
    } finally {
      setIsLoading(false);
    }
  };

  return { messages, isLoading, sendMessage, addMessage };
}
```

### Contrato da API — Endpoint de texto
- **URL:** `POST /api/atendimento/interagir`
- **Content-Type:** `application/json`
- **Body:** `{ Prompt, JsonHistory (JSON.stringify do array), PatientId, PatientName }`
- **Response:** `{ resposta: string }` → acessar via `response.resposta`
- **Implementação:** ver `src/services/api.ts` (criado na Story 1.1)

### Proteção de rota — `src/app/chat/page.tsx`

```tsx
'use client';

import { useEffect, useState } from 'react';
import { useRouter } from 'next/navigation';
import { getSession } from '@/lib/session';
import { PatientSession } from '@/types';

export default function ChatPage() {
  const router = useRouter();
  const [session, setSession] = useState<PatientSession | null>(null);

  useEffect(() => {
    const s = getSession();
    if (!s) { router.replace('/login'); return; }
    setSession(s);
  }, [router]);

  if (!session) return null; // aguarda redirect

  return <ChatInterface session={session} />;
}
```
